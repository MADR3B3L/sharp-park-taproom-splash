How this works

Make changes here, then click a yellow save button. If your browser supports direct saving, the update writes straight back to the site data. If not, the dashboard downloads the updated file for you to replace.

Best everyday workflow

  1. Open the live site in a second tab.
  2. Make a small change here.
  3. Save it.
  4. Refresh the live site to confirm it looks right.

Good to know

Words, taps, colors, vendor chips, and the floating vendor bubble are owner-editable. Bigger layout changes, photo galleries, or new sections can still be added later without breaking this dashboard.
